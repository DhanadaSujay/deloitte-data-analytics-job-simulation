# Deloitte Australia Data Analytics Job Simulation (Foarge)

This virtual job simulation involved completing practical data analytics tasks using Tableau and Microsoft Excel.

## Task 1: Tableau Dashboard
- Imported JSON telemetry data into Tableau.
- Created a calculated field called "Unhealthy".
- Built a "Down Time per Factory" bar chart.
- Built a "Down Time per Device Type" bar chart.
- Created an interactive dashboard where selecting a factory filters the device type chart.

## Task 2: Excel Analysis
- Added an "Equality Class" column.
- Categorized equality scores into:
    - Fair (-10 to +10)
    - Unfair (-20 to -11 amd +11 to +20)
    - Highly Discriminative (< -20 or > 20)
